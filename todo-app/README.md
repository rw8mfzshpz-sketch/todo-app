# NOTES ✦

Minimal, beautiful, and interactive task manager built with React.

## 🚀 Overview
Notes is a sleek, modern task management application designed for productivity. It features a cyberpunk-inspired aesthetic with smooth animations, multi-language support (English, Turkish, French), and local data persistence.

## ✨ Features
*   **Intuitive Task Management:** Add, toggle, edit, delete, and pin tasks.
*   **Priority Levels:** Categorize tasks by Low, Medium, or High priority with visual indicators.
*   **Tagging System:** Organize tasks with custom tags for better filtering.
*   **Due Dates:** Set and track deadlines with overdue and upcoming alerts.
*   **Multi-Language:** Fully localized interface (EN, TR, FR).
*   **Progress Tracking:** Visualize your productivity with a progress bar and status statistics.
*   **Persistence:** All data is securely saved to your browser's `localStorage`.
*   **Elegant UI:** A responsive, dark-themed design with smooth CSS animations and interactive elements.

## 🛠 Tech Stack
*   **React:** Frontend framework for building the UI components.
*   **TypeScript:** Ensuring type safety and better developer experience.
*   **Local Storage API:** For persistent data storage without a backend.
*   **CSS-in-JS:** Custom styles with CSS variables and keyframe animations for a polished look.

## 📦 How to Run
1.  **Clone the repository:**
    ```bash
    git clone https://github.com/yourusername/notes-app.git
    cd notes-app
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    ```
3.  **Start the development server:**
    ```bash
    npm run dev
    ```

## 🎨 Design Philosophy
This project focuses on creating a "minimal, beautiful" experience. The interface removes clutter, allowing users to focus on their tasks while providing powerful features like search, sorting, and categorization just one click away.
